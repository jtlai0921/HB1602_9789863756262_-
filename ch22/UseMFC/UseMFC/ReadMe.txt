========================================================================
    MICROSOFT 基礎類別庫 : UseMFC 專案概述
========================================================================


套用程式精靈已為您建立了此 UseMFC DLL。此 DLL 不僅示範 Microsoft 基礎類別的基本使用方法，還可作為您撰寫 DLL 的起點。

本檔案概要介紹群組成 UseMFC DLL 的每個檔案的內容。

UseMFC.vcproj
    這是使用套用程式精靈產生的 VC++ 專案的主專案檔案，
    其中包括產生該檔案的 Visual C++ 的版本訊息，以及有關使用套用程式精靈選取的平台、組態和專案功能的訊息。

UseMFC.h
    這是 DLL 的主頭檔案。它宣告了 CUseMFCApp 類別。

UseMFC.cpp
    這是主 DLL 源檔案。它包括 CUseMFCApp 類別。

UseMFC.rc
    這是程式使用的所有 Microsoft Windows 資源的清單。它內含 RES 子目錄中儲存的圖示、點陣圖和游標。此檔案可以直接在 Microsoft Visual C++ 中進行編輯。

res\UseMFC.rc2
    此檔案包括不在 Microsoft Visual C++ 中進行編輯的資源。您應該將不可由資源編輯器編輯的所有資源放在此檔案中。

UseMFC.def
    此檔案包括在 Microsoft Windows 中執行所必需的 DLL 的有關訊息。它定義了 DLL 的名稱和說明等參數，而且還從 DLL 匯出函數。

/////////////////////////////////////////////////////////////////////////////
其他標准檔案：

StdAfx.h, StdAfx.cpp
    這些檔案用於產生名為 UseMFC.pch 的預先編譯頭 (PCH) 檔案和名為 StdAfx.obj 的預先編譯型態檔案。

Resource.h
    這是標准頭檔案，可用於定義新的資源 ID。
    Microsoft Visual C++ 將讀取並更新此檔案。
